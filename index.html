Introduction

EVAReview is a utility program designed to assist in surgical planning and decision-making. It enhances healthcare quality through image segmentation and AI-based decision support, along with patient data management and secure communication tools.

System Architecture

EVAReview uses a U-Net convolutional neural network for biomedical image segmentation. The model is trained in Python with PyTorch and runs in the Node.js backend. Segmentation results are returned as JSON and displayed in the frontend.
